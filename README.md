# ViddlerThis is a repo for managing machine readable artifacts for the [Viddler APIs](http://www.viddler.com/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Viddler APIs](http://www.viddler.com/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://www.viddler.com/)- [Developer](http://www.viddler.com/)- [Twitter](https://twitter.com/viddler)- [GitHub](https://github.com/viddler)The [APIs.json](https://github.com/api-evangelist/viddler/blob/master/apis.json) index provides links to all the relevant links for the Viddler APIs, including the OpenAPI and Postman if they are present.
